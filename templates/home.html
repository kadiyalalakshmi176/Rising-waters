Workflow Stages

From raw historical meteorological parameters to deployable disaster mitigation classifiers.

01

Dataset

Historical data containing rainfall levels, temperature, humidity, and atmospheric visibility parameters.

02

Preprocessing

Audit check of missing values, IQR capping outliers, mapping categories, and feature scaling via StandardScaler.

03

Training

Comparative testing across Decision Trees, Random Forest, KNN, and XGBoost classifiers.

04

Model Selection

Selecting our optimized XGBoost configuration delivering exactly 96.55% validation accuracy.

05

Deployment

Joblib serialized objects deployed on IBM Cloud scalable environment ready for real-time inference requests.

Our selected model is configured with optimized hyper-parameters for tabular weather inputs:

  • Max Depth: 3 splits (combating overfitting)
  • Learning Rate: 0.1 step compression
